About the Project

About SlotAvailability

SlotAvailability is an independent data project built to make slot provider availability easier to understand worldwide.

The goal is simple: help users quickly check which slot providers are available by country and casino, without having to create accounts, manually browse casino lobbies, or search scattered forum posts.

What SlotAvailability tracks

The MVP currently tracks major slot providers such as Hacksaw Gaming, Pragmatic Play, Nolimit City, Relax Gaming, Push Gaming, Play'n GO, Evolution, BGaming, ELK Studios and Thunderkick across priority countries.

The first casino tracked is Stake. More casinos, countries and providers may be added as the project grows.

What SlotAvailability is not

SlotAvailability is not a casino, does not operate gambling services, does not accept deposits and does not provide games. It is an independent availability database and research tool.

Long-term vision

The long-term vision is to become a global reference for slot provider availability, covering more casinos, more providers, more countries, provider statistics, game-level data and an interactive world map.

If you want to help improve the project, you can report outdated data, suggest new casinos or support the project through the Help Us page.
